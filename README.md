## 项目简介

校园帮是一个面向在校大学生的校园互助平台，旨在解决学生日常生活中的零散需求。目标用户为校内学生，他们可以在平台上发布取快递、带饭、借书、辅导功课等求助任务，也可以接单帮助他人完成任务赚取虚拟币。

项目核心功能包括：用户注册登录、任务发布与接单、实时聊天沟通、任务完成确认、互评与信用分体系。通过虚拟币模拟交易，既避免了真实支付的复杂性，又能完整体现交易闭环。

### 技术栈（初步规划）

- **前端**：微信小程序
- **后端**：Node.js + Express框架（RESTful API，JWT用户认证）
- **数据库**：MySQL（存储用户、任务、评价等核心数据）
