## 遇到的问题和解决
1. **问题**：OpenAPI 文档参数格式与实际接口不一致，渲染报错
   **解决**：对照规范修正 schema 结构、必填项与枚举类型，确保文档可正常解析。

2. **问题**：前端请求时未正确携带 Token，导致接口 401 未授权
   **解决**：在请求拦截器中补充 Token 获取与 Authorization 头配置，统一处理登录态。

3. **问题**：任务列表分页、筛选参数较多，逻辑混乱
   **解决**：统一封装查询参数类型，支持 page/limit/status/type/sort 组合筛选，结构清晰可扩展。
