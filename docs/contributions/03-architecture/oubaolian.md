## 遇到的问题和解决
1. **问题**：初次绘制 Mermaid 架构图时语法不熟悉，图表渲染错误。  
   **解决**：查阅 Mermaid 官方文档，调整缩进和箭头方向，最终成功渲染。
2. **问题**：小程序开发中内容区滚动时出现顶部导航遮挡、底部栏文字透字问题。
   **解决**：采用「顶部操作区+内容展示区+底部导航区」三层布局，设置内容区独立垂直滚动，为内容区添加与底部栏等高的内边距，固定顶底栏高度，彻底解决显示问题。
3. **问题**：Figma设计稿样式转化为小程序SCSS时，部分配色、间距还原不一致。
   **解决**：提取设计稿中所有色值、字号、间距定义为全局SCSS变量，所有页面和组件统一调用变量，避免手动写死样式，保证设计稿1:1还原。
4. **问题**：多页面路由跳转时，底部TabBar与普通页面跳转逻辑冲突。
   **解决**：区分小程序`switchTab`（TabBar页面跳转）和`navigateTo`（普通页面跳转）API的使用场景，规范所有页面的跳转方式，解决路由冲突问题。
