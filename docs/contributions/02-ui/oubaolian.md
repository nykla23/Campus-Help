## 遇到的问题和解决
1. 问题：Figma原型中内容区滚动时遮挡顶部栏、底部栏透字
   解决：将页面拆分为“顶部操作区+内容展示区+底部导航区”三层结构，设置内容区独立垂直滚动，给内容区添加与底部栏等高的底部内边距，固定顶部/底部栏高度，彻底解决遮挡和透字问题。
2. 问题：使用v0.vercel生成的参考页面风格与校园帮定位不符，组件尺寸不适配移动端
   解决：参考v0.vercel生成的页面布局逻辑，保留核心交互思路，重新调整组件尺寸、色彩体系和字体层级，适配校园学生群体的使用习惯，确保设计风格贴合项目定位。
3. 问题：修改Figma颜色样式后，部分输入框/按钮颜色未同步更新
   解决：排查发现未同步元素为手动设置色值，未绑定全局样式，随后将所有元素统一绑定到色彩样式库，刷新预览缓存，实现样式一键同步。
