# 前端模块说明

## 1. 模块功能概述

前端模块是基于微信小程序开发的移动端应用，为校园互助平台“校园帮”提供用户交互界面。主要功能包括：

- **用户认证**：注册、登录、个人信息管理。
- **任务管理**：浏览任务列表、查看任务详情、发布任务、接单、确认完成任务、取消任务。
- **即时通讯**：基于腾讯云 IM 实现任务双方的实时聊天。
- **评价系统**：任务完成后双方互评，展示信用分。
- **个人中心**：查看个人资料、信用分、虚拟币余额、我发布的任务、我接单的任务、交易记录。



## 2. 技术选型

| 技术      | 版本/说明                                      |
| :-------- | :--------------------------------------------- |
| 开发框架  | 微信小程序                                     |
| UI 组件库 | Vant Weapp 1.10+                               |
| 即时通讯  | 腾讯云 IM SDK（最新版）                        |
| 状态管理  | 小程序全局数据（App.globalData）+ 本地缓存     |
| 网络请求  | 小程序原生 `wx.request`，封装请求拦截器        |
| 图片存储  | 腾讯云对象存储 COS（可选，用于头像、任务图片） |

## 4. 详细功能设计

### 4.1 用户认证模块

- **注册**：用户输入用户名、密码、学号（可选），前端进行格式校验（密码长度≥6，学号为数字），提交至后端，成功后自动登录并保存 token。
- **登录**：用户名+密码登录，获取 token 并存储到本地缓存。
- **身份保持**：在 `app.js` 中读取缓存 token，并挂载到全局数据，供后续请求使用。
- **退出登录**：清除本地缓存和全局数据，跳转至登录页。

### 4.2 任务模块

- **任务列表**：
  - 首页默认展示所有“待接单”任务，支持下拉刷新、上拉加载更多。
  - 顶部筛选栏：按状态筛选（待接单、进行中、已完成）。
  - 任务卡片组件展示标题、悬赏币、地点、发布者昵称和信用分。
  - 点击卡片跳转任务详情。
- **任务详情**：
  - 展示完整任务信息（标题、描述、悬赏、地点、截止时间、发布者信息、任务状态）。
  - 底部操作按钮根据任务状态和当前用户身份动态渲染：
    - 未登录：提示登录。
    - 发布者且待接单：可取消任务。
    - 发布者且进行中：可联系接单者，等待对方完成。
    - 发布者且待确认：可确认完成任务。
    - 非发布者且待接单：可接单。
    - 非发布者且进行中（当前用户是接单者）：可联系发布者，可放弃任务（可选）。
    - 任务完成后：可评价对方（若未评价）。
- **发布任务**：
  - 表单包括标题、描述、悬赏币、地点（可选）、截止时间（可选）。
  - 悬赏币不能超过当前账户余额，提交前校验。
  - 提交后跳转至任务详情页。

### 4.3 即时通讯模块

- **初始化**：在 `app.js` 中初始化腾讯云 IM SDK，登录后获取用户签名，生成 IM 登录凭证。
- **会话列表**：显示与任务相关的聊天会话，展示对方昵称、最后一条消息、未读计数。
- **聊天详情**：
  - 发送文字/图片消息（图片需先上传至 COS 获取 URL）。
  - 接收消息实时展示。
  - 聊天界面顶部显示任务标题，点击可跳转任务详情。
- **事件处理**：监听新消息、会话更新等事件，更新 UI。

### 4.4 评价模块

- **评价入口**：任务完成后，在任务详情页底部显示“去评价”按钮；个人中心的“待评价”列表也可进入。
- **评价表单**：选择星级（1-5），填写评语（可选），提交后更新任务状态并影响对方信用分。
- **评价展示**：在个人主页展示收到的评价列表，包括评价人、评分和评语。

### 4.5 个人中心模块

- **信息展示**：头像、昵称、学号（脱敏显示）、信用分、虚拟币余额。
- **我的发布**：列表展示当前用户发布的所有任务，可按状态筛选。
- **我的接单**：列表展示当前用户接单的所有任务。
- **交易记录**：虚拟币收入/支出流水，包含任务标题、金额、时间。
- **设置**：修改密码、退出登录。



## 5. 运行方式

### 5.1 环境准备

- 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号，获取 AppID

### 5.2 配置与启动

1. 克隆项目仓库，进入 `frontend` 目录。

2. 修改 `utils/config.js` 中的配置项：

   ```javascript
   export const API_BASE_URL = 'http://localhost:3000/api';  // 后端 API 地址
   export const IM_SDK_APP_ID = 123456789;  // 腾讯云 IM 应用 ID
   ```

3. 在微信开发者工具中导入 `frontend` 文件夹，填入自己的 AppID。

4. 点击“编译”即可在模拟器预览；如需真机调试，可使用预览码。
