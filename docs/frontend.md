# 前端模块说明

## 模块功能

- 用户注册/登录
- 任务发布、浏览、详情、接单、确认完成
- 实时聊天（基于腾讯云IM）
- 个人中心（资料、信用分、任务记录）

## 技术选型

- 框架：微信小程序原生开发
- UI组件库：Vant Weapp
- 即时通讯：腾讯云IM SDK
- 状态管理：小程序全局数据+缓存

## 目录结构

frontend/
├── pages/ # 页面
│ ├── index/ # 首页（任务列表）
│ ├── task/ # 任务详情
│ ├── publish/ # 发布任务
│ ├── user/ # 个人中心
│ └── chat/ # 聊天列表/详情
├── components/ # 公共组件
├── utils/ # 工具函数（请求封装、日期格式化等）
├── app.js # 小程序入口
├── app.json # 全局配置
└── project.config.json # 项目配置

## 运行方式

1. 下载安装微信开发者工具。
2. 克隆仓库后，用开发者工具打开 `frontend` 文件夹。
3. 在 `app.js` 中配置腾讯云IM的SDKAppID（如有）。
4. 点击“编译”即可在模拟器预览。
